# 贡献指南

感谢关注本项目！

## 开发流程

1. Fork 本仓库并克隆到本地
2. 创建分支：`git checkout -b feature/your-feature`
3. 安装依赖：`flutter pub get`
4. 修改代码并运行：`flutter test`、`flutter analyze`
5. 提交并发起 Pull Request

## 约定

- 保持红球 / 蓝球规则互不跨越
- 输入框需支持两位数连续输入，勿在输入过程中强制补零覆盖
- 重复号码应即时提示
- 提交信息尽量简洁说明「为什么改」

## 问题反馈

请通过 GitHub Issues 描述复现步骤、期望行为与实际行为。
